# Dostaffkin

Сервис оформления и отслеживания доставки посылок.

## Возможности
- Расчёт стоимости и срока доставки по адресу и параметрам посылки
- Оформление заявки на доставку с контактами
- Получение трек-номера
- Отслеживание статуса отправления по номеру

## Технологии
- Angular 21 (TypeScript)
- RxJS
- ngx-toastr (уведомления)
- REST API (https://testologia.ru/delivery)
- Яндекс.Карты (геокодинг, маршрутизация)

## Быстрый старт
```bash
npm install
npm start
```

## Структура
- src/app/pages/order — оформление доставки
- src/app/pages/track — отслеживание отправления
- src/app/services/delivery-api.ts — работа с API

## Скрипты
- `npm start` — запуск dev-сервера
- `npm run build` — сборка
- `npm test` — тесты
